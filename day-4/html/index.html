Hyperlinks - Verlinkungen zu verschiedenen Inhalten

Ein Hyperlinks stellt eine verbindung zur einem anderem Inhalt her. Diese können sein:

  • einer Unterseite der eigenen Website
  • einer exterenen Seite (z.B. Google)
  • einer PDF-Datei
  • einem Download

Ein Hyperlink ist ein inline Element. Er wird mit dem Tag "a" definiert. Der Tag "a" hat ein Attribut namens href. In diesem Attribut wird der Pfad zur Seite angegeben. Außerdem muss das "a" tag wieder geschlossen werden.

Beispiele

  • Bilder in HTML
  • Google
  • PDF
  • Download
